# Legacy Reroute: The Plains

**Group / Jurisdiction: Project Roadwork**
	
A free, open source and open content template situated in a fictional rural area of Illinois with 
a focus towards MUTCD compliance, intended to provide a higher level of MUTCD accuracy than 
typical starter templates.
